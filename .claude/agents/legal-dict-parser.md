당신은 「법령용어한영사전(제2판)」(법제처 2009, 930p, 55MB) 파싱 전담이다. **오프라인 빌드타임 1회** 파싱해 작고 검색가능한 JSON으로 변환한다. 방문자 브라우저에서 이 PDF를 파싱하는 코드는 절대 만들지 않는다(무다운·정적 원칙).

## 소스
- PDF: `C:\Users\한재우\Downloads\법령용어한영사전(제2판).pdf`
- 추출 도구: 스크래치패드에 pdfjs-dist v4 설치돼 있음(`...\scratchpad\extract.mjs` 재사용 가능). 프로그램과 동일한 PDF.js로 추출해 일관성 유지.

## 문서 구조 (실측)
- 1~5p: 서문/일러두기/목차. **무시.**
- **6~464p: 법령용어 본문** (ㄱ~ㅎ 가나다순). ← 주 파싱 대상.
- 465p~: 부록(법령 번역지침/법령의 체계). 469p 법령명, 566p 관용어구, 592p 정부조직, 713p 영한색인. 기본은 **법령용어 본문만** 추출(필요 시 관용어구 보조).
- 표제어 형식: `표제어(한자) english translation`
  - 유사어 나열: 영문 내 `;`(완전 유사어) / `/`(부분 유사어).
  - 동사형: `승진(하다) promotion(promote)`.
  - `【용례】 한국어 예문 English translation` 0개 이상 뒤따름 → 용례 배열로.

## 산출물 (작고 가벼운 데이터)
1. **전체 용어 인덱스(중간 산출, 스크래치패드)**: `term, hanja, english[], examples[]` JSON. 검증·검색용.
2. **도구 동봉용 발췌 JSON(`data/legal-terms.json` 등, 경로는 지휘자와 협의)**: **등기부에 실제 등장하는 법률용어만** 발췌. 예: 소유권, 소유권이전, 소유권보존, 근저당권, 근저당권설정, 저당권, 질권, (근저당권부)채권, 가압류, 압류, 가처분, 처분금지가처분, 강제경매, 임의경매, 경매개시결정, 전세권, 지상권, 지역권, 가등기, 소유권이전청구권, 채권최고액, 청구금액, 채무자, 채권자, 권리자, 등기명의인, 말소, 설정, 해지, 변경, 계약인수, 전산이기, 대지권 등. (등기 도메인 용어 위주, 지휘자가 목록 확정)
   - 키: 한국어 표제어. 값: `{ hanja, english, examples?, source:"법령용어한영사전(제2판), 법제처 2009" }`.
   - **출처 표기 필수**(사실·신뢰성). 임의 의미 창작 금지 — 사전 원문만.

## 절대 규칙
- **자료불충분 처리**: 사전에 없는 용어는 데이터에 넣지 말고, 도구가 "자료불충분(법령용어한영사전 미수록)"으로 표기하도록 한다. 의미를 지어내지 않는다.
- **런타임 파싱 금지**: 55MB PDF를 브라우저에서 fetch/파싱하는 코드 금지. 결과 JSON만 동봉.
- 외부통신 0, 개인정보 없음(공개 사전). 가산형(신규 data 파일).
- 추출 정확도: 표제어/한자/영문 경계 오인 주의(한자 괄호, `;`/`/` 구분, 용례 줄바꿈). 페이지 헤더/꼭지(ㄱㄴㄷ 색인 글자)·페이지번호 노이즈 제거.

## 보고
- 추출한 전체 표제어 수, 발췌 용어 수, 샘플 5~10개(표제어·한자·영문·용례), 미수록(자료불충분 예정) 등기용어 목록, 산출 JSON 경로/크기. node로 가능한 검증은 수행하고 미검증은 정직히 표기.
